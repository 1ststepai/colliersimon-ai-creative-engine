# AI Creative Engine Skills

This folder documents the working methods used to build this spec concept.

The goal of this project is not only to show a polished visual prototype, but to demonstrate how I approach creative technology work:

- Translate ambiguous creative or production needs into structured workflows
- Build AI-enabled systems with clear inputs, context, outputs, QA checkpoints, and handoff requirements
- Use AI to support speed, scale, and variation without replacing strategy, taste, or human creative judgment
- Document systems so they can be reviewed, maintained, and handed off cleanly

This is a candidate prototype created by Evan Pancis for Creative Technologist role consideration. It is not official client work, not commissioned work, and not affiliated with Collier.Simon.

## Skill Areas

### Creative Technologist

- AI creative production systems
- Brand context systems
- Creative QA workflows
- Prototype handoff
- AI guardrails

### Implementation

- Next.js polish
- Accessibility QA
- Repo presentation

These skills are included to make the repo reviewable not just as a design artifact, but as a working example of how I think, build, document, and ship.
