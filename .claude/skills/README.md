# Claude Skills

This folder contains Claude-oriented review skills for this repo.

These are intended to guide AI-assisted review, critique, and refinement of the prototype from the perspective of a Creative Technologist candidate.

## Skills Included

- Creative Technologist Review
- Context Engineering Review
- AI Output QA
- Handoff Documentation
- Repo Review
- Visual Polish Review

These skills complement the `.agents/skills` folder. The `.agents/skills` folder documents working methods and implementation standards. The `.claude/skills` folder documents review criteria and critique frameworks.
