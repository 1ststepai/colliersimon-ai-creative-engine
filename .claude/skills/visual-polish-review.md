# Claude Skill: Visual Polish Review

## Purpose

Use this skill to critique the visual quality of the prototype so it feels like a premium creative technology artifact, not a generic SaaS template.

## Review Criteria

### Visual Hierarchy

- Is the hero immediately strong?
- Are headings confident and readable?
- Is the layout scannable?
- Are important concepts emphasized?

### Agency Feel

- Does it feel relevant to a creative agency?
- Does it avoid generic dashboard clichés?
- Does it feel like a concept board plus internal prototype?
- Does it show taste without becoming over-designed?

### Interface Believability

The prototype console should feel like something a creative/production team could use.

Check for:

- Believable statuses
- Useful QA states
- Clear workflow stages
- Practical handoff details
- Structured content, not decorative filler

### Typography and Spacing

- Consistent spacing
- No cramped cards
- No clipped text
- Good line lengths
- Strong contrast
- Clean mobile stacking

### Final Standard

The design should make a reviewer think:

“This person can build a polished prototype and explain the system behind it.”
