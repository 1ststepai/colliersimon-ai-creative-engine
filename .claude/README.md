# Claude Project Guidance

This folder documents how Claude-style review and reasoning should be used on this spec concept.

The goal is to show that this prototype was not treated as a one-off visual demo. It was approached as a reviewable creative technology artifact with:

- Clear role alignment
- Context engineering
- Creative QA
- Accessibility review
- Repo hygiene
- Prototype handoff discipline
- Production-readiness thinking

This is a candidate prototype created by Evan Pancis for Creative Technologist role consideration. It is not official client work, not commissioned work, and not affiliated with Collier.Simon.

## How Claude Should Be Used

Claude should act as a senior creative technology reviewer across:

- Strategy fit
- Role alignment
- Systems clarity
- Copy clarity
- AI workflow logic
- Human-in-the-loop guardrails
- Design polish
- Documentation quality
- Handoff readiness

Claude should not fabricate facts, invent client data, or imply access to private agency information.
